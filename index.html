🔐 ciphers

Design a decoder ring, try it right here — drag the top disc to set the shift — then print it as a 2-page PDF and build the real thing.

shift 0

How to build it
  1. Print both PDF pages at 100% scale (no “fit to page”).
  2. Cut out both discs along the solid circles — plus the small window slot in the small disc.
  3. Place the small disc on the large one and push a split pin through both centre marks.
  4. Rotate the small disc to set the key — the window under the pointer shows the shift number.
  5. Encode: find each letter on the small disc, write the large-disc letter beside it. Decode: the reverse.
